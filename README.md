<td width="50%" valign="top">

**BBRES-RNG — Bit-Based Randomized Entropy System**

A custom pseudo-random number generator that harvests non-deterministic entropy by exploiting the unpredictable timing behaviour of the OS Thread Scheduler. The system employs a multi-threaded concurrency architecture in Java, aggregating system-level timing noise through bitwise mixing operations to produce high-quality random output.
